Sources tree
------------

Standart Maven project structure.
	|- src/ project sources
	     |- assembly/ file with maven-assembly-plugin assembly descriptor
	     |- etc/ file for maven-licence-plugin checks
	     |- main/ main sources
	     |- test/ tests sources
